在 macOS、Linux 和 WSL 上高效安裝與管理 Go 版本

本文運用 Golang 官方自 Go 1.11 引入並在 Go 1.20 版本穩定成熟的多版本管理機制，幫助開發者在 macOS、Linux 和 WSL 上高效安裝、升級與靈活切換 Go 版本。

2025-03-28

  • golang
  • manage
  • install
  • version

優勢

  1. 方便升級 Go 版本。
  2. 可安裝多個 Go 版本，方便測試或因應不同專案需要。

腳本限制

  1. CPU 只支援 x86-64 或 ARM64，作業系統只支援 macOS、Linux 和 WSL。其他的類 Unix 系統要自行調整 kernel 與 machine 變數，安裝包檔名列表可見 https://go.dev/dl/ 。
  2. Go 版本須大於 1.20，否則切換版本時的安裝路徑會有所不同。

列出可安裝的 Go 版本 (只選大於 1.20 的版本)

  1. 瀏覽 https://github.com/golang/dl

  2. 用命令列方式，需要 jq 指令

    1curl -s https://api.github.com/repos/golang/dl/contents \
2  | jq -r '.[] | select(.type=="dir") | .name' \
3  | grep "^go*" --color=never \
4  | sort -V

安裝第一個版本

先設定想安裝的版本，例如 1.24.1

1gover="1.24.1"

進行安裝

 1# Set nounset and errexit
 2set -ue
 3
 4# Set variables
 5extension="tar.gz"
 6kernel="$(uname -s | tr '[:upper:]' '[:lower:]')"
 7machine="$(uname -m)"
 8if [[ "$machine" = "x86_64" ]]; then
 9    machine="amd64"
10elif [[ "$machine" = "aarch64" ]]; then
11    machine="arm64"
12fi
13
14# Create directory for go installation
15mkdir -p "$HOME/sdk"
16
17# Download Go and install for the target version
18filename="go${gover}.${kernel}-${machine}.${extension}"
19curl -Lo "/tmp/${filename}" "https://go.dev/dl/${filename}" && \
20  tar -xzf "/tmp/${filename}" -C "$HOME/sdk" && \
21  rm "/tmp/${filename}"
22
23# Set the go version has been installed for Go wrapper
24touch "$HOME/sdk/go/.unpacked-success"
25
26# Move the installed go installation directory for Go wrapper
27mv "$HOME/sdk/go" "$HOME/sdk/go${gover}"
28
29# Install Go wrapper for this version
30"$HOME/sdk/go${gover}/bin/go" install "golang.org/dl/go${gover}@latest"
31
32# Check if the goVERSION command works
33"$HOME/go/bin/go${gover}" version
34
35# Set the goVERSION command to go command
36ln -sf "$HOME/go/bin/go${gover}" "$HOME/go/bin/go"

把 go 指令路徑加入 PATH 環境變數

設定 shell 啟動檔

執行自動化腳本

 1# Set nounset and errexit
 2set -ue
 3
 4# Determine the shell startup file
 5if [[ "$SHELL" == *bash ]]; then
 6    profile=~/.bash_profile
 7    prev_profile=~/.profile
 8elif [[ "$SHELL" == *zsh ]]; then
 9    profile=~/.zprofile
10    prev_profile=~/.zshrc
11else
12    echo "unknown shell, please set up your profile manually"
13    false
14fi
15
16# Create the profile if it doesn't exist
17if [[ ! -f "$profile" && -f "$prev_profile" ]]; then
18    echo -e "if [[ -f \"$prev_profile\" ]]; then\n  . \"$prev_profile\"\nfi" \
19      > "$profile"
20fi
21
22# Add go command path to env PATH
23gobinpath='export PATH="$HOME/go/bin:$PATH"'
24if [[ -z "$(command -v go)" ]] && \
25    ! grep -q "$gobinpath" "$profile"; then
26        echo -e "\n$gobinpath" >> "$profile"
27fi
28
29# Source the startup file
30set +u
31. "$profile"

或者手動加入 export PATH="$HOME/go/bin:$PATH" 進你的 shell 啟動檔 (shell startup file)。通常新版 Mac 使用 zsh，檔案為 ~/.zprofile；Linux 與舊版 Mac 通常使用 bash，檔案為 ~/.bash_profile，如果使用其他 shell 需要自行查詢。

檢查 go 指令是否生效

1go version

安裝另一個版本 (要修改 gover)

先設定想安裝的版本，例如 1.24.2

1gover="1.24.2"

進行安裝

 1# Set nounset and errexit
 2set -ue
 3
 4# Set variables
 5extension="tar.gz"
 6kernel="$(uname -s | tr '[:upper:]' '[:lower:]')"
 7machine="$(uname -m)"
 8if [[ "$machine" = "x86_64" ]]; then
 9    machine="amd64"
10elif [[ "$machine" = "aarch64" ]]; then
11    machine="arm64"
12fi
13
14# Install Go wrapper file for the target version
15go install "golang.org/dl/go${gover}@latest"
16
17# Download Go for the target version
18"go${gover}" download
19
20# Remove the downloaded archive file
21filename="go${gover}.${kernel}-${machine}.${extension}"
22rm -f "$HOME/sdk/go${gover}/${filename}"
23
24# Set the goVERSION command to go command
25ln -sf "$HOME/go/bin/go${gover}" "$HOME/go/bin/go"

檢查 go 指令是否生效

1go version

切換 Go 版本

列出已安裝的 Go 版本

1ls "$HOME/sdk" | grep "^go" --color=never

切換成指定的 Go 版本

先設定到想切換的目標版本，例如 1.24.2

1gover="1.24.2"

進行切換

1set -u
2ln -sf "$HOME/go/bin/go${gover}" "$HOME/go/bin/go"

檢查版本切換是否成功

1go version

移除已安裝的 Go 版本

列出已安裝的 Go 版本

1ls "$HOME/sdk" | grep "^go" --color=never

先把 go 指令指向要保留的版本

先設定要保留的版本，例如 1.24.2

1gover="1.24.2"

進行切換

1set -u
2ln -sf "$HOME/go/bin/go${gover}" "$HOME/go/bin/go"

檢查版本切換是否成功

1go version

移除指定的 Go 版本

先設定你想移除的版本，例如 1.24.1

1gover="1.24.1"

進行移除

1set -ue
2rm -rf "$HOME/sdk/go${gover}"
3rm "$HOME/go/bin/go${gover}"
